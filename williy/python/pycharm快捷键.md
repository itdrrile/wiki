## pycharm快捷键

#### 1 多行代码同时缩进

​    鼠标选中多行，按下Tab键，一次缩进4个字符

#### 2 多行代码同时左移

​    鼠标选中多行代码，同时按住shift + Tab键，一次左移四个字符

#### 3 设置用鼠标滚动调整字体大小

​    File->Setting->General->mouse 向第二个复选框打勾，这样就可以ctrl + 鼠标滚动的方式来调整字体    大小

### 4 编辑类

* ctrl + / 行注释
* ctrl + shift + ]/[ 选定代码块结束、开始
* shift+Tab 不缩进当前行
* Tab 缩进当前行
* ctrl+D 复制选定的区域或行
* ctrl + Y 删除选定的行

#### 5 导航类

* ctrl + N 跳转到类
* ctrl + shift + N 跳转到文件
* alt + right/left 跳转到下一个、前一个编辑的选项卡
* ctrl + G 查看当前行号、字符号
* ctrl + alt + left/right 后退、前进
* ctrl + alt + B 跳转到实现
* ctrl + shift + B 跳转到类型声明
* ctrl + F12 弹出文件结构
* ctrl + H 类型层次结构
* ctrl + shift + H  方法层次结构
* ctrl + Alt + H 调用层次结构

#### 6 查找/替换类

* F3 下一个
* shift + F3 前一个
* ctrl + R 替换
* ctrl + shift + F 全局查找
* ctrl + shift + R 全局替换

#### 7重构类

* shift + F6 文件重命名
